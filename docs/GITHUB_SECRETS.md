# Required GitHub Secrets

Go to: GitHub Repository → Settings → Secrets and variables → Actions

## Required Secrets

| Secret Name | Description |
|---|---|
| `RENDER_DEPLOY_HOOK` | Get from Render dashboard: Service → Settings → Deploy Hook → Copy URL |
| `RENDER_APP_URL` | e.g. `https://scalekit-api.onrender.com` |
| `CODECOV_TOKEN` | Token provided by codecov.io after connecting your repository |

## Optional Secrets (for notification integrations)

| Secret Name | Description |
|---|---|
| `SLACK_WEBHOOK_URL` | Incoming webhook URL for posting build notifications to Slack |

## Auto-provided Secrets

| Secret Name | Description |
|---|---|
| `GITHUB_TOKEN` | Automatically generated by GitHub Actions, used for GHCR push |
